Configuração do Access Point

Configure as credenciais do ponto de acesso WiFi do ESP32

Informações Importantes:

  • O ESP32 criará uma rede WiFi com estes dados
  • A senha deve ter pelo menos 8 caracteres
  • O IP deve estar na faixa válida (ex: 192.168.x.x)
  • Após salvar, o ESP32 será reiniciado
